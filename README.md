# Veridian Paint Co. Website

Refined static site + favicons + PWA manifest.

## Deploy on GitHub Pages
1. Create a public repo named `veridianpaint.github.io` and upload the CONTENTS of this folder.
2. Settings → Pages → Source: `main` → `/ (root)`.
3. Custom Domain: `veridianpaint.com` (this repo has a CNAME already).
4. In Squarespace DNS, set A records: 185.199.108.153 / 109.153 / 110.153 / 111.153 and CNAME `www` → `veridianpaint.github.io`.
5. Enable HTTPS (Enforce).
